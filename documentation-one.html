Instalasi

(Cara memasang template Blogspot.)

Setelah membeli template, Anda akan menerima zip file berisi: file XML, file dokumentasi dan lisensi.

  1. Extrack file zip.
  2. Cari file XML.
  3. Buka file XML dengan Notepad atau code editor lainnya.
    Klik kanan >> open with >> Notepad.
    (Jangan buka file XML dengan Ms. Word).
  4. File tampak seperti ini:
  5. Copy semua file. CTRL + A >> CTRL + C
  6. Masuk ke dashboard Blogspot.
  7. Dashboard >> Theme >> Klik dropdown >> pilih "Edit HTML".
  8. Anda akan dibawa ke tampilan seperti ini:
Setelah template terpasang, beberapa widget akan mengikuti template demo. Jadi, Anda perlu untuk setting ulang atau mengganti beberapa bagian agar sesuai dengan blog Anda.
